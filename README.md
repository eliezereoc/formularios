# template-driven
Este projeto foi gerado com [Angular CLI](https://angular.io/cli) version 13.3.3.
 
## Formulário de cadastro com requisição a API VIACEP

> Dependencies

    -Angular material  
    -Flex-layout
    -Bootstrap

Install the dependencies...

```bash
cd template-driven
npm install
```

Then start

```bash
ng serve
```
